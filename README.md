# Jiangxi-fish-function
Data and script for analyzing the fish species functional diversity in the Jiangxi

## Repository structure

- input/
  - Raw or initial datasets used for analysis.

- data_processed/
  - Processed datasets generated from input data.

- scripts/
  - R scripts used for data processing and statistical analyses.

- output/
  - Figures and tables generated by the scripts.

Analysis workflow:

input → data_processed → scripts → output
